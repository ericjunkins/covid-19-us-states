Interpreting the Covid-19 Trajectory chart

This chart is to show the growth rate of COVID-19 on a scale that that makes seeing if it is spreading at an exponential rate, or if the curve has begun to "flatten". It is important to note that it is on a Logarithmic scale, and also that the "Y-axis" is the total cases in the last 7 days, meaning it will be more conservative in showing trends.

  • Exponential growth line: Given by x=y, this line denotes where exponential growth in cases is occuring
  • A vertical line would indicate that there was no new cases of COVID-19, that is the eventual goal
  • Look for significant deviation from the exponential growth line , with significant downward trends to see a state experiencing "flattening the curve"
  • Stay at Home legislative orders issued by each state goverment mark the number of total cases a state had

Inspiration of this visulization from https://aatishb.com/covidtrends/ and MinutePhysics

Interpreting the Severity Bubbles

This chart is to show the relative severity of each state when its local government decided to take legislative action. It is made to be sortable/filterable by different criteria so you can use many variables to determine if a state had an appropriate response to the pandemic.

The Severity of a state is given by the radius of each individual bubble

The Y-axis Grouping of a state filters how the bubbles are positioned on the Y-axis, given different filtering criteria

The Time Since specific event of a state is given by the color of the bubble, less time appearing more white, and longer appearing more red

Interpreting the Legislative Orders Chart

This chart is to show the legislative actions taken by the local government, sorted by which type of order it is and dates it was released on

The fainter rectangles in the chart are the number of that type of order given on each day. More order types will come in the future.

This chart can be used to correlate the Date of the order by state to the data in the line chart above to determine if trends emerge in response to this event, either spikes/resurgenses in cases, or "flattening" of the curve

COVID-19 US States Data

Explore the US historical data for coronavirus, and how it relates to each state's government issued orders

About

This dashboard is meant to help inform the user about how each of the US states are doing in response to the Covid-19 crisis, both in an absolute sense and relative sense. It is meant to be interactive to aid in seeing relationships in the data that are otherwise difficult to come about. The main data source comes from Covid Tracking Project
